# कैलकुलेटर (Calculator)

Create a calculator using kalaam

```
इनपुट(FirstValue)

इनपुट(SecondValue)

इनपुट(Operation)


प्रिंट('FirstValue is '+FirstValue)
प्रिंट('SecondValue is '+SecondValue)




अगर(Operation=='+')

{

output=FirstValue+SecondValue

प्रिंट("Output is " + output)

}

अगर(Operation=='-')

{

output=FirstValue-SecondValue

प्रिंट("Output is " + output)

}

अगर(Operation=='*')

{

output=FirstValue*SecondValue

प्रिंट("Output is " + output)

}

अगर(Operation=='/')

{

output=FirstValue/SecondValue

प्रिंट("Output is " + output)

}
```
